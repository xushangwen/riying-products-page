# 日盈产品页面原型开发需求

## 项目目标
使用 HTML + CSS + JS 开发产品展示页面的交互原型，实现三级产品分类导航系统。

## 技术要求
- **技术栈**：原生 HTML + CSS + JavaScript（无框架）
- **响应式**：Desktop/Tablet/Mobile 三端适配
- **样式**：现代化设计，使用 Flexbox/Grid 布局
- **图标**：Remix Icon v4.9.1
- **字体**：中文 IBM Plex Sans SC / 数字 Space Grotesk / 英文 Google Sans

## 页面层级结构

### 一级页面（产品分类总览）
- 展示 4 个大类标题
- 每个大类下方以 **Grid 卡片形式** 展示该大类的所有二级小类
- 卡片可点击进入二级页面

### 二级页面（小类产品列表）
- 顶部显示当前大类标题
- 以 **Grid 卡片形式** 展示该小类下的所有三级具体产品
- 卡片可点击进入三级页面

### 三级页面（产品详情）
- 产品名称
- 产品描述
- 产品特点（列表形式）

## 开发要点

### 数据管理
- 使用 JSON 格式存储产品数据，便于动态渲染
- 数据与页面逻辑分离，便于后续添加产品

### 路由逻辑
- 使用 URL Hash 或 History API 实现页面切换
- 支持浏览器前进/后退

### 样式规范
- 卡片设计现代化、简洁
- 悬停效果（hover）提升交互体验
- 移动端优化触摸交互
- 使用相对单位（rem/vw）确保响应式

### 性能优化
- 图片使用 lazy loading
- 优先使用 WebP/AVIF 格式

### 可访问性
- 语义化 HTML 标签
- 键盘导航支持
- 适当的 ARIA 标签
